Requisitos

Todo lo que necesitas saber para adquirir tu licencia de conducción.

Edad

  • ✔ Tener 16 años, si es para conducir vehículos de uso particular,moto A2 o B1.
  • ✔ Tener 18 años, si es para conducir vehículo de servicio público C1.
  • ✔ No tener comparendos.
  • ✔ A1: Autoriza la conducción de motocicletas con un cilindraje igual o menor a 125 c.c.
  • ✔ A2: Para motocicletas, motociclos y mototriciclos con cilindraje superior a 125 c.c (incluye las motos de la categoría A1).

R.U.N.T

  • ✔ Debes estar inscrito en el registro único nacional de tránsito (R.U.N.T) para obtener tu licencia de conducción. Esta es la BASE DE DATOS oficial que contiene la información de todos los conductores, vehículos y licencias en Colombia.

Examen Médico

  • ✔ Haber aprobado el examen de aptitud física, mental y de coordinación motriz para conducir vehículos de la categoría que aspira a obtener la licencia. Este certificado debes realizarlo en CRC VALORAR CONDUCTORES.
